Un módulo en Python es un archivo con extensión .py que contiene código reutilizable:
funciones, clases o variables.
Actúa como una biblioteca para organizar la lógica,
permitiendo importar estas funcionalidades 
en otros scripts mediante import para mejorar la estructura,
legibilidad y modularidad del software.
